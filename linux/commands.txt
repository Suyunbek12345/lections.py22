CTRL+ALT+T - команда открытия терминала
MacOS(В спотлайт вводим Terminal)

~ - символ домашней дериктории(папки)
/ - символ корневой корневой директории(папки)


. . - обознаечается предыдущую директории
. - обозначает текущую дирекцию

1. ls(list) -> Выводит список всех директории и файлов в текущей директории(рабочая директория)

ls - a -> выводит список как и скрытых так и обычных файлов и директории

ls -l ->  выводит список директории и файлов с подробной информацией

2. cd(change directory) - Ю менят текущую директорию на указанную
cd < название директории>

3. mkdir(make directory) -> Содает указанную директорию
mkdir <директория>

4. touch -> создает указанный файл
touch <name_of_file>.<extension(расширение (txt, docs))>

5. pwd(print working directory) -> выводит путь до текущей директории

6. rmdir (remove directory) -> удаляет указанную директорию(не может удалить не пустую директорию)

7. rm -> удвление файла
 rm <name_of_file>
 
rm -rf -> принудительное удаление файла либо директории

8. mv ->   перемещает либо переименовывает файл либо директорию, в зависимости от указанного второго аргумента

mv <файл_директория> <новое_имя> -- директория/>

9. cat -> выводит содержимое указанного файла 

10. nano ->    открывает графический интерфейс в терминале для редактирвания файла
* если нет указанного файла, то создаться новый файл с таким именем

11. echo ->  команда для быстрой записи данных в файл через терминал

echo текст для записи > название файла

12. cp (copy) - команда для копирования файла в нужную директорию
cp <file_to_copy> <path>

nano <name_of_file>

CTRL+SHIFT+C  копировать в терминалеcd``lstouc
CTRL+SHIFT+V вставить в терминале
